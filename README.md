# Expenses_Analyzer_Streamlit


A simple **personal finance tracker dashboard** built using Python.  
It helps analyze expenses from an Excel file and visualizes spending patterns using interactive charts.

---

## 🚀 Features

- Load expense data from Excel
- Clean and preprocess data automatically
- Category-wise expense grouping
- Pie chart visualization of spending
- Simple Streamlit dashboard interface

---

## 🛠️ Tech Stack

- Python 🐍  
- Pandas 📊  
- Matplotlib 📉  
- Streamlit ⚡  
- OpenPyXL 📁
